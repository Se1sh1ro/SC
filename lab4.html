Лабораторная работа №4

Соответствует лекции: Лекция №3 — Управление случайностью: temperature и top_p

Цель: Освоить влияние параметров temperature и top_p на разнообразие и креативность ответов модели, научиться выбирать значения в зависимости от типа задачи.

1
Активируйте виртуальное окружение и перейдите в папку `claude_labs`.

7

8
Выведите значение температуры и три полученных названия.
9
Запустите скрипт и проанализируйте результаты.
10
Создайте новый файл `temperature_recipes.py`.
11
Создайте промпт: `"Придумай рецепт салата из простых ингредиентов"`.
12
Выполните запрос с `temperature=0.2` и зафиксируйте результат.

Сравните логичность и креативность обоих рецептов.

27
Сравните правильность ответов при разной температуре.

Сохраните все результаты в отчёт.
import os from dotenv import load_dotenv import anthropic load_dotenv() client = anthropic.Anthropic(api_key=os.getenv("ANTHROPIC_API_KEY")) print("=== Тест temperature ===\n") for temp in [0.0, 0.3, 0.5, 0.8, 1.0]: print(f"Температура = {temp}") for i in range(3): response = client.messages.create( model="claude-3-haiku-20240307", max_tokens=50, temperature=temp, messages=[{"role": "user", "content": "Придумай необычное название для кофейни"}] ) print(f" {i+1}: {response.content[0].text}") print() print("=== Рецепт при низкой температуре (0.2) ===") response = client.messages.create( model="claude-3-haiku-20240307", max_tokens=300, temperature=0.2, messages=[{"role": "user", "content": "Придумай рецепт салата из простых ингредиентов"}] ) print(response.content[0].text) print("\n=== Рецепт при высокой температуре (0.8) ===") response = client.messages.create( model="claude-3-haiku-20240307", max_tokens=300, temperature=0.8, messages=[{"role": "user", "content": "Придумай рецепт салата из простых ингредиентов"}] ) print(response.content[0].text) print("\n=== Код при низкой температуре (0.1) ===") response = client.messages.create( model="claude-3-haiku-20240307", max_tokens=300, temperature=0.1, messages=[{"role": "user", "content": "Напиши функцию на Python для сортировки списка"}] ) print(response.content[0].text)
